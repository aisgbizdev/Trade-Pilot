## STEP 1 — MAIN ANALYSIS VIEW (DEFAULT)

Display:

1. Market Condition
   - trend / range / volatile

2. Directional Bias
   - Beginner:
     "Cenderung naik / Cenderung turun / Tunggu dulu"
   - Pro:
     "Bullish bias / Bearish bias / Neutral"

   DO NOT use BUY / SELL

3. Invalidation (HIGH PRIORITY — MUST BE NEAR TOP)
   Display clearly:
   "This analysis becomes invalid if:"
   - condition 1
   - condition 2

   (use existing invalidationConditions field)

4. Scenario Breakdown
   - Scenario A (dominant)
   - Scenario B (alternative)
   - Scenario C (wait / no-trade)

5. Opportunity vs Risk
   - Show relative upside vs downside
   - Keep neutral tone

6. Confidence Range
   - Example: 52%–60%
   - Include reason from `uncertaintyNotes`
   - Explain why it is not certain

7. Explanation
   - Beginner: simple, guided
   - Pro: structured + technical + fundamental drivers
