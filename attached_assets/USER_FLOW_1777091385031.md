# User Flow

## Beginner User Flow
1. User opens app
2. User signs in
3. User selects Beginner Mode
4. User chooses instrument and timeframe
5. User clicks Analyze
6. App shows:
   - market condition summary
   - risk level
   - confidence range
   - possible direction scenarios
   - why this view exists
   - what can make it fail
7. User saves result to history

## Pro User Flow
1. User opens dashboard
2. User selects Pro Mode
3. User inputs more detailed context
4. User clicks Analyze
5. App shows:
   - base case
   - bullish case
   - bearish case
   - confidence
   - invalidation conditions
   - volatility/risk context
   - explanation
6. User saves and compares with past analyses
