# Product Requirements Document

## Product Name
AI Trading Assistant MVP

## Goal
Create a responsive web app optimized for mobile view that helps beginner and advanced traders interpret market conditions using AI-assisted analysis.

## User Segments
1. Beginner users
2. Advanced/pro users

## Main Modes
### Beginner Mode
- Simple language
- Show market condition summary
- Show risk meter
- Show confidence range
- Show short reason
- Show warning when uncertainty is high
- Show educational notes

### Pro Mode
- Detailed analysis
- Multiple scenarios
- Confidence score
- Probability distribution style explanation
- Invalidation conditions
- More technical reasoning

## Core Features for MVP
1. Landing page
2. Login/register
3. Dashboard
4. Toggle between Beginner and Pro mode
5. Analysis form:
   - instrument
   - timeframe
   - market notes / optional price context
6. AI analysis result card
7. Analysis history
8. Feedback button:
   - useful / not useful
9. Admin-lite page for prompt/config editing (optional if easy)

## Non-Goals for MVP
- No broker integration
- No autotrading
- No payment gateway
- No social feed
- No advanced chart engine in v1
- No custom ML training pipeline in v1

## Design Direction
- Clean dark modern UI
- Premium but simple
- Mobile-first responsive layout
- Fast loading
- Big cards and simple tab navigation

## Safety and UX Rules
- Do not present output as guaranteed prediction
- Do not say guaranteed profit or guaranteed win rate
- Always show uncertainty and risk context
- Always include “when this analysis may fail”
